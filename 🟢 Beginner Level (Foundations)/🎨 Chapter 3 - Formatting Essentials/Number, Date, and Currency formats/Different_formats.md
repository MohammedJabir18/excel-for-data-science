# Excel Cell Formatting Types

Here are the common Excel cell formatting types with examples:

## 1. General
- Default format
- Example: `1234.567` displays as `1234.567`

## 2. Number
- Displays numbers with specified decimal places
- Example: `1234.567` with 2 decimal places → `1234.57`

## 3. Short Date
- Displays date in short format
- Example: `1/15/2023` or `15/01/2023` (locale-dependent)

## 4. Long Date
- Displays date with weekday and month name
- Example: `Sunday, January 15, 2023`

## 5. Currency
- Displays value with currency symbol
- Example: `$1,234.57` (negative values in red or parentheses)

## 6. Accounting
- Aligns currency symbols and decimal points
- Example: `$ 1,234.57`

## 7. Time
- Displays time values
- Example: `3:30:45 PM`

## 8. Percentage
- Displays value as percentage
- Example: `0.75` → `75%`

## 9. Fraction
- Displays decimal as fraction
- Example: `0.5` → `1/2`

## 10. Scientific
- Displays in scientific notation
- Example: `1234.567` → `1.23E+03`

## 11. Text
- Treats content as text (no calculation)
- Example: `00123` remains `00123` instead of `123`

## 12. Custom
- User-defined formats using format codes
- Examples:
  - `000-00-0000` for SSN
  - `###.##` for variable decimal places
  - `"Text: "@` to prepend text
